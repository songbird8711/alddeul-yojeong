# 테스트 실행 방법

핵심 로직(카테고리 분류, 쇼핑리스트 매칭, 계산, 월간 예산 등)이 잘 동작하는지
확인하는 테스트입니다. 실제 배포되는 `js/` 폴더의 코드를 그대로 읽어서 실행하기
때문에, 여기서 통과했다면 실제 코드도 같은 방식으로 동작한다는 뜻입니다.

(카메라, PaddleOCR, OpenCV처럼 실제 브라우저/폰이 있어야만 확인 가능한 부분은
이 테스트로 검증되지 않습니다 — 그런 부분은 실기기에서 직접 확인해야 합니다.)

## 실행법

```bash
cd tests
npm install
npm test
```

## 코드를 고칠 때마다

새 기능을 추가하거나 기존 로직을 고칠 때, 관련된 테스트를 `run.js`에 같이
추가/수정해두면 나중에 실수로 뭔가를 망가뜨렸을 때 바로 알 수 있습니다.
(예: "새우깡이 수산으로 오분류되던 버그"처럼, 한 번 잡은 버그는 테스트로
남겨두면 같은 실수가 재발하는 걸 막을 수 있습니다.)
