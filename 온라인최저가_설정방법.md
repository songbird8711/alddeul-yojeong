# 온라인 최저가 기능 설정 방법

이 기능은 네이버쇼핑 검색 API를 사용합니다. 아래 순서대로 하면 됩니다 (전부 무료).

## 1단계. 네이버 개발자센터에서 API 키 발급받기

1. https://developers.naver.com 접속 → 네이버 계정으로 로그인
2. 상단 메뉴에서 **Application → 애플리케이션 등록** 클릭
3. 아래처럼 입력:
   - 애플리케이션 이름: 알뜰요정 (아무 이름이나 가능)
   - 사용 API: **검색** 체크
   - 비로그인 오픈 API 서비스 환경 → **WEB 설정** 선택
   - 웹 서비스 URL: Vercel에 배포한 실제 주소 입력 (예: `https://알뜰요정이름.vercel.app`)
4. 등록 완료하면 **Client ID**와 **Client Secret**이 발급됩니다. 이 두 값을 복사해두세요.

## 2단계. Vercel에 환경변수로 등록하기

1. https://vercel.com 대시보드 → 알뜰요정 프로젝트 클릭
2. **Settings → Environment Variables** 이동
3. 아래 두 개를 추가:
   - Key: `NAVER_CLIENT_ID` / Value: (1단계에서 받은 Client ID)
   - Key: `NAVER_CLIENT_SECRET` / Value: (1단계에서 받은 Client Secret)
4. 저장 후, **Deployments** 탭에서 최신 배포를 다시 배포(Redeploy)해야 환경변수가 적용됩니다.

## 3단계. 확인

1. 앱에서 상품명(예: "국산 삼겹살 500g")을 입력하고 비교하기
2. 결과 화면에서 "🔍 온라인 최저가 확인" 버튼 클릭
3. 실제 최저가 목록이 뜨면 성공입니다.

## 참고

- 네이버 API는 하루 25,000회까지 무료입니다. 개인 사용 목적으로는 충분합니다.
- 상품명을 구체적으로 입력할수록(브랜드명, 용량 포함) 검색 결과가 정확해집니다.
- API 키를 설정하지 않으면 "🔍 온라인 최저가 확인" 버튼을 눌렀을 때 안내 메시지가 뜨고, 나머지 기능(단가 비교 등)은 그대로 정상 작동합니다.
