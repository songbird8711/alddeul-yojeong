# 알뜰요정 배포 방법 (Vercel)

코드는 이미 완성되어 있습니다. 아래 순서대로 하면 실제 URL로 배포됩니다.

## 방법 A. 웹사이트에서 드래그로 배포 (가장 쉬움, 5분)

1. https://vercel.com 접속 → 회원가입(깃허브/구글 계정으로 가능)
2. 로그인 후 대시보드에서 **"Add New..." → "Project"** 클릭
3. 화면에 "Import Git Repository" 대신 아래쪽의 **"또는 폴더를 여기로 드래그하세요"** 영역에
   압축 해제한 `alddeul-yojeong` 폴더 전체를 끌어다 놓습니다.
   (Vercel은 순수 HTML/CSS/JS 프로젝트는 빌드 설정 없이 바로 인식합니다)
4. "Deploy" 클릭 → 1분 내로 `https://알뜰요정이름.vercel.app` 같은 주소가 생성됩니다.
5. 이 주소를 폰 브라우저로 열고 "홈 화면에 추가"를 누르면 앱처럼 아이콘이 생깁니다.

## 방법 B. 커맨드로 배포 (터미널 사용 가능하다면)

```
npm install -g vercel
cd alddeul-yojeong
vercel
```
질문이 나오면 전부 기본값(Enter)으로 진행하면 됩니다. 완료되면 URL이 출력됩니다.

## 이후 수정하고 싶을 때

- 코드를 수정한 뒤 같은 폴더에서 `vercel --prod` 를 다시 실행하면 즉시 반영됩니다.
- 또는 Vercel 대시보드에서 프로젝트를 깃허브 저장소와 연결해두면,
  깃허브에 코드를 올릴 때마다 자동으로 재배포됩니다.

## 배포 후 꼭 확인할 것 (검증 체크리스트)

- [ ] 폰에서 접속 시 레이아웃이 깨지지 않는지
- [ ] 상품 A/B에 값을 입력하면 실시간으로 100g당/개당 가격이 뜨는지
- [ ] "비교하기" 클릭 시 영수증 결과와 저렴한 쪽에 도장이 찍히는지
- [ ] 새로고침해도 "최근 비교 기록"이 남아있는지 (localStorage 확인)
- [ ] 비행기모드(오프라인)에서도 페이지가 열리는지 (서비스워커 캐싱 확인)
- [ ] 홈 화면에 추가 시 알뜰요정 아이콘이 뜨는지
